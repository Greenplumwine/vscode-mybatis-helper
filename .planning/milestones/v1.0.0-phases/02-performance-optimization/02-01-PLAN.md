# Plan 02-01: Cache Optimization

## Objective
Implement cache optimization strategies: scheduled cleanup for FastMappingEngine indexes and two-level cache for RegexUtils.

## Background
Current issues:
1. FastMappingEngine uses multiple Map indexes with no eviction policy
2. RegexUtils has unbounded regex cache growth

This plan implements the decisions from discuss-phase: scheduled cleanup and two-level cache.

## Task 1: Add scheduled cleanup to FastMappingEngine

<read_first>
- src/features/mapping/fastMappingEngine.ts
- src/features/mapping/indexCache.ts (reference LRU pattern)
</read_first>

<action>
Add scheduled cleanup mechanism to FastMappingEngine:

1. Add cleanup timer and configuration:
   ```typescript
   // In FastMappingEngine class
   private cleanupInterval: NodeJS.Timeout | null = null;
   private readonly DEFAULT_CLEANUP_INTERVAL = 30 * 60 * 1000; // 30 minutes
   ```

2. Add cleanup control methods:
   ```typescript
   /**
    * Start scheduled cleanup timer
    */
   public startCleanupTimer(intervalMs: number = this.DEFAULT_CLEANUP_INTERVAL): void {
     if (this.cleanupInterval) {
       clearInterval(this.cleanupInterval);
     }
     this.cleanupInterval = setInterval(() => {
       this.cleanupStaleEntries();
     }, intervalMs);
     this.logger?.debug(`Started cleanup timer with ${intervalMs}ms interval`);
   }

   /**
    * Stop cleanup timer
    */
   public stopCleanupTimer(): void {
     if (this.cleanupInterval) {
       clearInterval(this.cleanupInterval);
       this.cleanupInterval = null;
       this.logger?.debug('Stopped cleanup timer');
     }
   }
   ```

3. Implement stale entry cleanup:
   ```typescript
   /**
    * Remove entries for files that no longer exist or have been modified
    */
   private async cleanupStaleEntries(): Promise<number> {
     let removed = 0;
     const fs = await import('fs/promises');

     for (const [namespace, index] of this.namespaceIndex) {
       try {
         // Check if Java file still exists
         if (index.javaPath) {
           await fs.access(index.javaPath);
         }
         // Check if XML file still exists
         if (index.xmlPath) {
           await fs.access(index.xmlPath);
         }
       } catch {
         // File no longer exists, remove from indexes
         this.removeMapping(namespace);
         removed++;
       }
     }

     if (removed > 0) {
       this.logger?.info(`Cleaned up ${removed} stale mappings`);
     }
     return removed;
   }
   ```

4. Update `initialize()` to start cleanup timer:
   ```typescript
   public async initialize(): Promise<void> {
     const { Logger } = await import('../../utils/logger.js');
     this.logger = Logger.getInstance();
     this.startCleanupTimer();
   }
   ```

5. Add dispose method for cleanup:
   ```typescript
   /**
    * Clean up resources when extension deactivates
    */
   public dispose(): void {
     this.stopCleanupTimer();
     this.removeAllListeners();
   }
   ```
</action>

## Task 2: Implement two-level regex cache

<read_first>
- src/utils/performanceUtils.ts (current RegexUtils implementation)
</read_first>

<action>
Replace unbounded regex cache with two-level cache:

1. Update RegexUtils class structure:
   ```typescript
   export class RegexUtils {
     private static instance: RegexUtils;

     // Two-level cache
     private hotCache: Map<string, RegExp> = new Map();
     private coldCache: Map<string, RegExp> = new Map();

     private readonly HOT_CACHE_SIZE = 50;
     private readonly COLD_CACHE_SIZE = 50;

     // Cache statistics
     private stats = {
       hotHits: 0,
       coldHits: 0,
       misses: 0
     };

     private constructor() {}

     public static getInstance(): RegexUtils {
       if (!RegexUtils.instance) {
         RegexUtils.instance = new RegexUtils();
       }
       return RegexUtils.instance;
     }
   ```

2. Implement two-level cache retrieval:
   ```typescript
   /**
    * Get regex from cache or create new one
    * Uses two-level cache: hot (frequently used) and cold (previously used)
    */
   public getRegex(pattern: string, flags?: string): RegExp {
     const key = flags ? `${pattern}|${flags}` : pattern;

     // Check hot cache first
     const hotEntry = this.hotCache.get(key);
     if (hotEntry) {
       this.stats.hotHits++;
       return hotEntry;
     }

     // Check cold cache (promote to hot if found)
     const coldEntry = this.coldCache.get(key);
     if (coldEntry) {
       this.stats.coldHits++;
       this.promoteToHot(key, coldEntry);
       return coldEntry;
     }

     // Create new regex
     this.stats.misses++;
     const regex = new RegExp(pattern, flags);
     this.addToHotCache(key, regex);
     return regex;
   }
   ```

3. Implement cache management:
   ```typescript
   /**
    * Add regex to hot cache, evicting if necessary
    */
   private addToHotCache(key: string, regex: RegExp): void {
     // If hot cache is full, move oldest to cold cache
     if (this.hotCache.size >= this.HOT_CACHE_SIZE) {
       const firstKey = this.hotCache.keys().next().value;
       if (firstKey !== undefined) {
         const firstRegex = this.hotCache.get(firstKey)!;
         this.hotCache.delete(firstKey);
         this.addToColdCache(firstKey, firstRegex);
       }
     }

     this.hotCache.set(key, regex);
   }

   /**
    * Add regex to cold cache, evicting if necessary
    */
   private addToColdCache(key: string, regex: RegExp): void {
     // If cold cache is full, evict oldest
     if (this.coldCache.size >= this.COLD_CACHE_SIZE) {
       const firstKey = this.coldCache.keys().next().value;
       if (firstKey !== undefined) {
         this.coldCache.delete(firstKey);
       }
     }

     this.coldCache.set(key, regex);
   }

   /**
    * Promote cold cache entry to hot cache
    */
   private promoteToHot(key: string, regex: RegExp): void {
     this.coldCache.delete(key);
     this.addToHotCache(key, regex);
   }
   ```

4. Add cache statistics method:
   ```typescript
   /**
    * Get cache statistics
    */
   public getStats(): {
     hotSize: number;
     coldSize: number;
     hotHits: number;
     coldHits: number;
     misses: number;
     hitRate: number;
   } {
     const totalHits = this.stats.hotHits + this.stats.coldHits;
     const total = totalHits + this.stats.misses;
     return {
       hotSize: this.hotCache.size,
       coldSize: this.coldCache.size,
       hotHits: this.stats.hotHits,
       coldHits: this.stats.coldHits,
       misses: this.stats.misses,
       hitRate: total > 0 ? totalHits / total : 0
     };
   }

   /**
    * Clear all caches
    */
   public clearCache(): void {
     this.hotCache.clear();
     this.coldCache.clear();
     this.stats = { hotHits: 0, coldHits: 0, misses: 0 };
   }
   ```
</action>

<acceptance_criteria>
- RegexUtils has hot cache (50 entries) and cold cache (50 entries)
- `getRegex()` checks hot cache first, then cold cache
- Cold cache hits are promoted to hot cache
- Hot cache evictions move to cold cache
- Cold cache evictions are removed
- `getStats()` returns cache statistics including hit rate
- `clearCache()` clears both caches and resets stats
- `pnpm run compile` passes
- `pnpm run test:unit` passes (existing tests still work)
</acceptance_criteria>
